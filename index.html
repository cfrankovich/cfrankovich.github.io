current project

As of today (07/01/2022) I am working on a programming language called MASP. MASP stands for the macro and statistical programming. With this name in mind, my goal is to create an interpreted language similar to lisp that focuses on data management and analysis.

Once version 1.0 releases, I will write a blog explaining my thought process and code. See the project in its current state at github.com/cfrankovich/MASP.
