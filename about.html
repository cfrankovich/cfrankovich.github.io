CARSON(7)

NAME

carson frankovich (i use arch btw)

SYNOPSIS

carson [OPTION]... [FILE]...

DESCRIPTION

carson is a terminal tool developed in the early 2000s available on Linux and Windows 10. carson efficiently works with Vim and Arch Linux. The preferred device that carson should run on is a Thinkpad x220 but could also run on other hardware.

-a, --arch

recommended distro to run carson on

-bw

activate bedwars sweat mode

-C/C++

carson works best with reading and writing C/C++ code

-m, --music

usually runs better when listening to vaporwave or punk music

-p, --python

can read and write python code

-t, --thinkpadx220

carson's recommended and most used hardware

-u, --ultimate

option used for a fun time outdoors. forehand throws still are buggy
